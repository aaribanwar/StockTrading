# StockTrading

## Live sites
- Frontend: https://frontendstocktrading.netlify.app/
- Dashboard: https://dashboardstocktrading.netlify.app/
- Backend: https://backendstocktrading.onrender.com/

## Stack
- Frontend: React, Bootstrap
- Dashboard: React
- Backend: Node.js, Express, MongoDB

## Run locally (backend)
1. cd backend
2. npm install
3. export MONGODB_URI="your_mongo_uri"
4. npm start

## Notes
- Ensure backend CORS allows frontend origins.
- Health endpoint: GET /health
